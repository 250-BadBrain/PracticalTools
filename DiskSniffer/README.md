# DiskSniffer

一个 Windows 原生磁盘占用可视化工具。

## 功能

- 启动后选择盘符或输入路径
- 后台扫描文件和文件夹大小
- 使用矩形树图显示磁盘占用
- 鼠标悬停矩形时显示完整路径和大小
- 左键单击文件夹进入该文件夹继续扫描
- 右键单击文件或文件夹打开系统右键菜单
- 按 `Backspace` 返回上一个扫描根，返回根盘符时会优先使用缓存
- 支持设置显示展开深度
- 支持设置最小占比，过小的目录会合并显示以减少视觉噪声
- 扫描卷根目录时会显示无法枚举的系统/受保护空间

## 使用

运行程序，选择盘符或输入路径后点击 `Scan`。

参数：

```text
Path：相对于当前盘符的路径，留空表示扫描盘符根目录
Depth：最大展开深度，深层内容仍会计入父级大小
Min %：相对扫描根目录的最小占比，小于该比例的目录会合并显示
```
