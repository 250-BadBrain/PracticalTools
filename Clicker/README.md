# 连点器

一个轻量的 Windows 连点器小工具，可用于自动执行鼠标点击或按键操作。

## 功能

- 支持鼠标左键、右键、中键连点
- 支持键盘按键连发
- 可设置每秒点击次数
- 支持单一操作模式和组合操作模式
- 支持全局快捷键启动、停止和退出
- 自带窗口图标，可直接运行 `clicker.exe`

## 使用

双击运行 `clicker.exe`，在窗口中选择点击类型、速度和模式后启动即可。

常用快捷键：

- `F6`：开始或停止
- `F8`：退出程序

## 文件

- `clicker.exe`：可直接运行的程序
- `clicker.cpp`：源代码
- `clicker.rc` / `app.ico`：程序图标资源
