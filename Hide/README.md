# Locker

一个用于隐藏本地 `Locker` 文件夹的 Windows BAT 小工具。

## 功能

双击 `locker.bat` 即可管理文件夹：

- 第一次运行：创建 `Locker` 文件夹。
- 当 `Locker` 可见时：询问是否隐藏。
- 当 `Locker` 已隐藏时：输入密码后恢复显示。

把需要隐藏的文件放进 `Locker`，然后再次运行 `locker.bat` 即可隐藏。

## 密码

默认密码是：

```text
1234
```

如需修改密码，编辑 `locker.bat` 顶部这一行：

```bat
set "PASSWORD=1234"
```

## 注意事项

- 这是一个轻量隐藏工具，不是加密工具。
- 适合在 Windows 本机做简单的文件隐藏。
- 重要文件请自行做好备份。
